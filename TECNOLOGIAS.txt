================================================================================
PADRÕES ARQUITETURAIS
================================================================================

  - SPA (Single Page Application)    Frontend React servido pelo Vite (porta 3000)
  - REST API                         Backend FastAPI com endpoints JSON (porta 8000)
  - ORM + Migrations                 SQLAlchemy + Alembic
  - Task Queue                       Celery + Redis para processamento assíncrono
  - JWT Authentication               Token Bearer em localStorage, expiração 8h
  - Circuit Breaker                  Padrão de resiliência para APIs externas
  - Retry + Backoff                  tenacity em todas as chamadas às APIs externas
  - Fail-Graceful                    Fonte offline retorna null; outras fontes continuam
  - RPA                              Playwright para automação bancária
  - Idempotência                     UniqueConstraint em proposta_id_externo
  - Vite Proxy                       Dev server em :3000 proxia :8000 (elimina CORS)

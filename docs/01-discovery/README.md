# Discovery - Sistema Antifraude

## Objetivo

Esta fase tem como objetivo compreender o processo atual de aprovação de propostas, identificar problemas operacionais, entender as necessidades do negócio e registrar dúvidas antes do início da definição da solução.

## Escopo desta fase

* Entendimento do processo atual (AS IS)
* Levantamento das dores operacionais
* Identificação dos participantes envolvidos
* Mapeamento de oportunidades de melhoria
* Registro de dúvidas e pontos a validar

## Contexto

Atualmente a empresa utiliza o sistema WebDeck para realizar importação de propostas bancárias, validações antifraude, aplicação de regras de aprovação e integração com robôs responsáveis pela aprovação das propostas nos bancos.

O objetivo do projeto é desenvolver uma solução própria capaz de reproduzir e evoluir esse processo, reduzindo dependência de fornecedores externos e aumentando o controle sobre as regras de negócio.

# Perguntas em Aberto

## Parceiros

1. Um parceiro pode pertencer a mais de uma esteira simultaneamente?
2. Existe hierarquia entre esteiras?
3. Quem possui permissão para alterar esteiras?

## Regras

4. Todas as regras atuais precisam ser reproduzidas?
5. Existem regras específicas por banco?
6. Existem regras específicas por convênio?

## Aprovação

7. Quais condições obrigam análise manual?
8. Existe limite máximo para aprovação automática?
9. Existem exceções autorizadas pela operação?

## Blacklist

10. Quem pode cadastrar registros?
11. Quem pode remover registros?
12. Existe aprovação para remoção da blacklist?

## Convênios

13. Todo convênio deve ser criado automaticamente?
14. Existem convênios que devem ser bloqueados?
15. Existe classificação de risco por convênio?

## Integrações

16. Quais bancos serão priorizados na primeira versão?
17. Quais bancos possuem API?
18. Quais bancos dependem exclusivamente de RPA?

## Monitoramento

19. Quais indicadores operacionais são considerados críticos?
20. Qual o tempo máximo aceitável para importação de propostas?
21. Qual o tempo máximo aceitável para aprovação automática?

## Projeto

22. Qual será o MVP da primeira versão?
23. Qual processo obrigatoriamente deve estar disponível no go-live?
24. Quais funcionalidades podem ficar para versões futuras?
